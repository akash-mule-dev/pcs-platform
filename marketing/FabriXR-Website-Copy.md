## 2. Features page

**Page headline:** Everything you need to run the floor — and nothing you don't.
**Intro:** FabriXR covers production execution, live labor, and 3D/AR quality. Here's what's live today.

**Live floor visibility**
Know who's on which order and stage, against target, in real time. The dashboard surfaces work orders, active operators, completed output and labor efficiency, with a Live Stage Status board (operator · order · stage · elapsed · station) that updates with no refresh, on web and mobile.

**Real-time labor tracking**
Operators clock into a specific stage of a specific work order in one tap. Idle, break and rework are captured, not guessed. Supervisors can review and correct entries, and compare actual vs. target time by stage — the accurate labor data that finally makes job costing and quoting possible.

**3D/AR quality & assembly inspection**
Inspect a built assembly against the actual 3D model and pin defects to the geometry — type, severity, measurements, tolerances — with AR-assisted inspection at the station. Sign-offs and defect-pattern views build a real quality record. *(Deeper scan-to-model comparison is in active development.)*

**Work orders & kanban**
A real lifecycle: draft → pending → in-progress → completed, with priorities, due dates, line assignment, order dependencies and batch operations — plus a kanban board to move work the way the floor moves.

**Process & routing templates**
Model each product's routing once: ordered stages with target times, versioned and drag-to-reorder. The same part gets built the same way every time, and actual-vs-target is comparable by stage.

**Operator mobile app**
Assigned work, one-tap clock in/out, 3D/AR reference, and an offline queue for the dead zones of the floor. iOS and Android.

**Roles, access & audit**
Role-based access for managers, supervisors and operators, with an audit log of who changed what.

**Built to fit your stack**
An Angular web portal, a React Native operator app, and a documented REST + real-time (WebSocket) API — deployed on standard cloud infrastructure.

## 6. FAQ

- **What is FabriXR?** A cloud production-control platform for metal-fab and discrete shops: real-time floor visibility, per-stage labor tracking, and 3D/AR quality inspection, on web and an operator app.
- **Who is it for?** Metal-fabrication and sheet-metal job shops, roughly 20–300 employees, running multi-stage routed jobs.
- **How long does it take to go live?** Weeks, not quarters. We model your routings during onboarding and start with a pilot line.
- **Does it replace our ERP?** No — it's the live execution-and-quality layer your floor will actually use. It can run alongside an ERP. Materials, scheduling and costing are on our roadmap.
- **What's the 3D/AR part?** Operators inspect built assemblies against the actual 3D model and pin defects to the geometry, with AR viewing at the station.
- **Web or mobile?** Both — a web portal for managers and supervisors, an operator app (iOS/Android) for the floor, one live data source.
- **Is there an API?** Yes, a documented REST + real-time (WebSocket) API.
